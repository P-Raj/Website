CollabEdit (Source)

CollabEdit is a P2P realtime collaborative editing tool for eclipse. It allows multiple users connected via a local network to work on the same code.Because it does not involve any central server and runs on the local networkit ensures privacy and productivity. It was developed as a part of our(me and Swetank Kumar Saha) Distributed Systems project.
